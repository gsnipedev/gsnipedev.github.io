Relay Controller (https://github.com/gsnipedev/GSnipeRelay2Controller)
Aplikasi desktop yang saya buat menggunakan C#, .NET versi 4, dan Windows Form library adalah sebuah solusi yang platform-independent untuk mengontrol input Relay Hardware pada sistem Windows.

Aplikasi ini memberikan antarmuka yang intuitif dan mudah digunakan, memungkinkan pengguna untuk mengendalikan relay hardware dengan mudah dan efisien. Dengan menggunakan C# dan .NET versi 4, aplikasi ini memiliki kinerja yang handal dan dapat diandalkan.

Melalui penggunaan Windows Form library, aplikasi ini menyediakan tampilan grafis yang menarik dan responsif, sehingga pengguna dapat dengan mudah mengakses dan mengontrol relay hardware.

Catatan: Aplikasi ini ditujukan untuk developer, yang akan dimodifikasi sebagai library sehingga dapat terintegrasi ke software yang ia kembangkan.
Fitur:
  1. Fitur Relay Control pada umumnya
  2. Support hingga 8 input
  3. Dapat digunakan di Windows 7.
  4. Copy-Paste-able code, sejalan dengan tujuan software ini dibuat.
